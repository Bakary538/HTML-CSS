C -Direction

Flexbox nous permet d'agencer les elements dans le sens que l'on veut. Avec flex-direction, on peut les positionner verticalement ou encore les inverser. Il peut prendre les valeurs suivantes:
- row: organisé sur une ligne (vleurs par defaut)
- column: orgenisé sur une colonne
- row-reverse: organisé sur une ligne, mais en ordre inversé
- column-reverse: organisé sur un cologne, mais en ordre inverse

E-Alignement

Les elements sont organisés soit horizontalemnt (par defaut)soit verticalement cela definit ce qu'on appelle l'axe principale. Il y a aussi un axe secondaire(cross axis):
-> si vos elements sont organisés horizontalemntl'axe secondaire est l'axe verticale.
-> si vos elements sont organisés verticalement, l'axe secondaire est l'axe horizontale.

Alignement sur l'axe principal

Afin d'aligner nos elements sur l'axe principal: justify-content
elle peut prendre comme valeur:
-> flex-start: aligner au debut (par defaut)
-> flex-end: aligner a la fin
-> center: aligner au centre
-> space-between: les elements sont etirés sur tout l'axe (il y a de l'espace entre eux)
-> space-around: les elements sont etirés sur tout l'axe, mais ils laissent aussi de l'epace sur les extremités
-> space-evenly: Les elements sont repartis uniformement dans le conteneur d'alignement le long de l'axe principal.

F - Repartir plusieurs lignes

Si vous avez plusieurs lignes dans votre flexbox vous pouvez choisir commen celle-ci seront repartis avec align-content. Cette propriété permet de gerer l'alignement des differents lignes (ou colonnes) les uns par rapport aux autres. Cette propiété n'aura d'effet que si le conteneur flexible possedes plusieurs lignes (si l'axe principale est horizontal) ou plusieurs colonnes (si l'axe principale est vertical) et que si le flex-wrap est present Les valeurs de cette propriété:
->flex-start : les elements sont placés au debut
->felx-end : les elements sont placés a la fin
->center : les elements sont placés au centre
->space-between : les element sont séparés avec de l'espace entre eux
->space-around : les element sont separés avec de l'espace entre eux, mais il y a aussi des espaces au debut et a la fin
->stretch : la valeurs par defaut : les elements s'etirent pour occuper tout l'espace -> space-evenly: Les elements sont repartis uniformement dans le conteneur d'alignement le long de l'axe principal.
